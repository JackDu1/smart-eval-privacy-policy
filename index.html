Privacy Policy for Smart Eval Assistant

Effective Date: March 2026

Welcome to Smart Eval Assistant (hereinafter referred to as "the Extension"). We deeply understand the extreme importance of student and user data privacy. The Extension strictly complies with relevant laws and regulations and provides services based on the principles of "minimum necessary, authorized flow, and security compliance."

1. Data Source and Authorization

The Extension DOES NOT use web crawlers or scraping to intercept sensitive privacy data from irrelevant web pages. All core archive data and certificate information are securely acquired via authorized Server-to-Server APIs from official certificate service platforms.

2. Information We Process

  • Authorized Certificate Data: Desensitized factual dimensions (e.g., certificate level, award time) retrieved securely via authorized tokens.
  • Authentication Info: Session tokens used to verify your current access permissions.
  • Service Logs: API call counts and status logs for quota management, which are not linked to any student's personal identity.

3. Volatile Processing & Data Retention

The data we process is used solely to provide contextual cues for the underlying Large Language Model (AI) to generate objective evaluations. Once the AI generation is complete, all temporary data is immediately destroyed from our server's memory. We absolutely do not build or retain any student privacy databases.

4. Third-Party Sharing

We transmit desensitized text to enterprise-grade AI model providers for generation purposes only. Our AI partners are bound by strict enterprise data protection agreements, ensuring your data is never used to train public AI models.

5. Security

All data transmission is forced over HTTPS (TLS 1.2/1.3). Your local authorization tokens are securely stored in the browser's sandbox and are instantly erased upon uninstalling the Extension.

“综评智能小助手”隐私政策

生效日期： 2026年3月

欢迎使用“综评智能小助手”（以下简称“本插件”）。我们深知学生及用户数据隐私的极端重要性。本插件严格遵循相关法律法规，秉持“最小够用、授权流转、安全合规”的原则提供服务。

1. 核心声明：数据来源与授权前提

本插件不会通过网页爬虫或监听手段抓取非相关网页上的敏感隐私数据。本插件使用的所有证书信息及核心数据，均通过安全的后端 API 接口（Server-to-Server）获取，且已提前获得官方平台的合法授权。

2. 我们处理哪些信息？

  • 授权证书数据：通过安全接口定向拉取的已脱敏事实维度（基于 Token 校验）。
  • 账户鉴权信息：用于校验您当前使用权限的临时会话密钥。
  • 基础服务日志：用于保障并发与计费的系统调用次数，不与学生隐私挂钩。

3. 信息的流转与“阅后即焚”机制

获取的数据仅用于提供给大模型（AI）生成评语。云端服务器在返回生成的评语后，所有涉及个体的临时档案数据将立即从服务器内存中销毁。我们绝不建立、私自保存任何隐私数据库。

4. 我们与谁共享信息？

我们会将剔除身份标识后的脱敏文本发送给提供算力的企业级 AI 平台。我们承诺：对接的 AI 服务商已签署严格的数据保护协议，传入的 API 数据绝不会被用于训练公共模型。

5. 传输与本地安全

全链路强制采用 HTTPS (TLS 1.2/1.3) 加密传输。您的授权凭证仅加密存储于浏览器本地，卸载插件后将立即安全擦除。
